  • named(_:)

    Looks up a hash function algorithm by name (e.g., sha256). Uses OpenSSL’s EVP_get_digestbyname function.

    let algorithm = try DigestAlgorithm.named("sha256")

    Throws

    CryptoError if no digest for that name is found.
