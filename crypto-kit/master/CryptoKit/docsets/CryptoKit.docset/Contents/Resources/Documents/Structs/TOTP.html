TOTP

public struct TOTP

Generates Time-based One-time Passwords using HMAC.

let code = TOTP.SHA1.generate(secret: "hi")
print(code) "123456"

You can also generate ranges using generateRange(...).

  • generateRange(degree:digits:secret:at:)

    Generates a range of TOTP tokens to a specific degree. This method calls the generate(...) method internally.

    let codes = try TOTP.SHA1.generateRange(degree: 1, secret: key)
print(codes) // [String]

    Declaration

    Swift

    public func generateRange(degree: Int, digits: OTPDigits = .six, secret: CryptoData, at date: Date = .init()) throws -> [String]

    Parameters

    degree

    Number of codes (in addition to the main code) to generate in both the forward and backward direction. This number must be at least 1. For each degree, the total code count will be increased by two: one for an additional degree in the positive and negative offset directions. For example, if degree is 2, there will be 5 codes returned: The main code, two codes at offset 1 (1 and -1), and two codes at offset 2 (2 and -2).
    digits

    Number of digits to include in the password. Defaults to six.
    secret

    Cleartext (not Base32 encoded) secret key.
    date

    Date to generate the TOTP for. This will be divided into intervals automatically.
