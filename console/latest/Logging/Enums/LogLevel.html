  • verbose

    Verbose logs are used to log tiny, usually irrelevant information. They are helpful when tracing specific lines of code and their results

    Example:

    let a = 1 + 4
logger.debug(a)
let b = 1 - 3
logger.debug(b)
let c = (b + a) * 3
logger.debug(c)
