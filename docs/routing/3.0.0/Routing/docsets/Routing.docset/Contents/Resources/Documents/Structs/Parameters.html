Parameters

public struct Parameters

Holds resolved Parameter values that are discovered while routing.

After this struct has been filled with parameter values, you can use it to fetch them out in order using the next(...) method.

let id = parameters.next(Int.self)
  • values

    The stored ParameterValues. These can be converted into their associated Parameters using the next(...) method.

  • next(_:on:)

    Grabs the next parameter from the parameter bag.

    Note: the parameters must be fetched in the order they appear in the path.

    For example GET /posts/:post_id/comments/:comment_id must be fetched in this order:

    let post = try parameters.next(Post.self, on: ...)
let comment = try parameters.next(Comment.self, on: ...)
