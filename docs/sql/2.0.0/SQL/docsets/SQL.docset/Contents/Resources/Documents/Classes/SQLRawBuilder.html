SQLRawBuilder

public final class SQLRawBuilder<Connection>: SQLQueryBuilder, SQLQueryFetcher
    where Connection: SQLConnection

Builds raw SQL queries.

conn.raw("SELECT * FROM planets WHERE name = ?")
    .bind("Earth")
    .all(decoding: Planet.self)

  • bind(_:)

    Binds a single encodable value to the query. Each bind should correspond to a placeholder in the query string.

    conn.raw("SELECT * FROM planets WHERE name = ?")
    .bind("Earth")
    .all(decoding: Planet.self)

    This method can be chained multiple times.

  • binds(_:)

    Binds an array of encodable values to the query. Each item in the array should correspond to a placeholder in the query string.

    conn.raw("SELECT * FROM planets WHERE name = ? OR name = ?")
    .binds(["Earth", "Mars"])
    .all(decoding: Planet.self)
