ContentContainer

public struct ContentContainer<M> where M : HTTPMessageContainer

Helper for encoding and decoding Content from an HTTP message.

req.content.decode(User.self)

See Request and Response for more information.

    Parses a Decodable type from this HTTP message. This method supports streaming HTTP bodies (chunked) and can run asynchronously. See syncDecode(_:) for the non-streaming, synchronous version.

    This method accepts a custom JSONDecoder. See JSONDecoder.custom(...) for a convenient way to create a customized instance.

Sync
  • syncDecode(_:)

    Parses a Decodable type from this HTTP message. This method does not support streaming HTTP bodies (chunked) and runs synchronously. See decode(_:maxSize:) for the streaming version.

    let user = try req.content.syncDecode(User.self)
print(user) // User

    The HTTP message’s MediaType will be used to lookup the relevant HTTPBodyDecoder to use.

    Throws

    Any errors making the decoder for this media type or parsing the message. An error will also be thrown if this HTTP message’s body type is streaming.
