TagRenderer

public protocol TagRenderer

Capable of rendering instances of TemplateTag in the AST. Each invocation of a tag in a template will result in a call to the identified TagRenderer with a TagContext containing the current context.

  • render(tag:)

    Renders a TemplateData to return to the TemplateSerializer for serialization.
