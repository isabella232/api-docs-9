  • bootstrap(console:level:metadata:)

    Bootstraps a ConsoleLogger to the LoggingSystem, so that logger will be used in Logger.init(label:).

    Declaration

    Swift

    public static func bootstrap(console: Console, level: Logger.Level = .info, metadata: Logger.Metadata = [:])

    Parameters

    console

    The console the logger will log the messages to.
    level

    The minimum level of message that the logger will output. This defaults to .debug, the lowest level.
    metadata

    Extra metadata to log with the message. This defaults to an empty dictionary.
