Console

Adds the ability to run Commands on a Console.

  • run(_:input:on:)

    Runs a CommandRunnable (CommandGroup or Command) of commands on this Console using the supplied CommandInput.

    try console.run(group, input: &env.commandInput, on: container).wait()

    The CommandInput will be mutated, removing any used CommandOptions and CommandArguments. If any excess input is left over after checking the command’s signature, an error will be thrown.

    CommandGroup or Command to run.

    Mutable CommandInput to parse CommandOptions and CommandArguments from.

    Return Value

    A Future that will complete when the command finishes.
