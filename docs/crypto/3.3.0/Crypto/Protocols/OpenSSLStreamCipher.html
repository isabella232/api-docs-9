OpenSSLStreamCipher

public protocol OpenSSLStreamCipher

OpenSSLStreamCipher is a protocol representing a higher-level interface for managing various OpenSSL stream ciphers.

  • reset(key:iv:mode:) Default implementation

    Resets / initializes the cipher algorithm context. Must be called before calling update. See the default implementation extension on this protocol for more.

    Default Implementation

    Resets / initializes the cipher algorithm context. This must be called once before calling update(data:)

    let key: Data // 16-bytes
var aes128 = Cipher(algorithm: .aes128ecb)
try aes128.reset(key: key, mode: .encrypt)

    Throws

    CryptoError if reset fails, data conversion fails, or key/iv lengths are not correct.

  • update(data:into:) Default implementation

    Encrypts or decrypts a chunk of data into the supplied buffer. See the default implementation extension on this protocol for more.

    Default Implementation

    Encrypts or decrypts a chunk of data into the supplied buffer.

    let key: Data // 16-bytes
let aes128 = Cipher(algorithm: .aes128ecb)
try aes128.reset(key: key, mode: .encrypt)
var buffer = Data()
try aes128.update(data: "hello", into: &buffer)
try aes128.update(data: "world", into: &buffer)
print(buffer) // Partial ciphertext

    Note: You must call reset() once before calling this method.

    Throws

    CryptoError if update fails or data conversion fails.

  • finish(into:) Default implementation

    Finalizes the encryption or decryption, appending any additional data into the supplied buffer. See the default implementation extension on this protocol for more.

    Default Implementation

    Finalizes the encryption or decryption, appending any additional data into the supplied buffer.

    let key: Data // 16-bytes
let aes128 = Cipher(algorithm: .aes128ecb)
try aes128.reset(key: key, mode: .encrypt)
var buffer = Data()
try aes128.update(data: "hello", into: &buffer)
try aes128.update(data: "world", into: &buffer)
try aes128.finish(into: &buffer)
print(buffer) // Completed ciphertext

    Note: You must call reset() once and update() at least once before calling this method.

    CryptoError if finalization fails.
