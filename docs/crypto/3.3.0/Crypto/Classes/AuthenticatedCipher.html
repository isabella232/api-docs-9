AuthenticatedCipher

public final class AuthenticatedCipher : OpenSSLStreamCipher

AuthenticatedCipher supports AEAD-type ciphers. It feels a lot like ‘Cipher’ except that it supports the AEAD tag and related validation.

  • encrypt(_:key:iv:)

    Encrypts the supplied plaintext into ciphertext. This method will call reset(key:iv:mode:), update(data:into:), and finish(into:) automatically.

    let key: Data // 32-bytes
let iv: Data // 12-bytes
let (ciphertext, tag) = try AES256GCM.encrypt("vapor", key: key, iv: iv)
print(ciphertext) /// Encrypted Data
print(tag) /// GCM authentication tag

    Throws

    CryptoError if reset, update, finalization or tag retrieval steps fail or if data conversion fails.

  • decrypt(_:key:iv:tag:)

    Decrypts the supplied ciphertext back to plaintext. This method will call reset(key:iv:mode:), update(data:into:), and finish(into:) automatically.

    let key: Data // 32-bytes
let iv: Data // 12-bytes
let (ciphertext, tag) = try AES256GCM.encrypt("vapor", key: key, iv: iv)
try AES256GCM.decrypt(ciphertext, key: key, iv: iv, tag: tag) // "vapor"

    Throws

    CryptoError if reset, update, or finalization steps fail or if data conversion fails.
