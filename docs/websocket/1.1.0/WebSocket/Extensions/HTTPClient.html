HTTPClient

final class HTTPClient

Allows HTTPClient to be used to create WebSocket connections.

let ws = try HTTPClient.webSocket(hostname: "echo.websocket.org", on: ...).wait()
ws.onText { ws, text in
    print("server said: \(text)")
}
ws.send("Hello, world!")
try ws.onClose.wait()

    hostname

    Remote server’s hostname.

    Remote server’s port, defaults to 80 for TCP and 443 for TLS.
    path

    Path on remote server to connect to.
    headers

    Additional HTTP headers are used to establish a connection.
    maxFrameSize

    Maximum WebSocket frame size this client will accept.
    worker

    Worker to perform async work on.
